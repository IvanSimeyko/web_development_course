Проверка поддерки браузером той иои иной возможнжности canIuse.com or html5please.com

Для распознавания js-кода браузеры используют виртуальные машины или движки, которые интерпретируют и выполняют код на js. У каждого браузера своя такая машина.

Самый популярный браузер Chrom, второй, как ни стронно IE.

Структура браузера:
1. юзер интерфейс
2. browser engine
3. rendering engine - читает, анализирует, отображает интерент страницу.
4.1 networking - обеспечивает связь интернет строницы с сервером
4.2. javascript interpreter - виртуальная машина, которая выполняет js код
4.3. backend - тут понятно
5 data storage - место на локалной машине где браузер сохраняет данные. 

Якорь # - предназначен для внутренней адресации по ресурсу.

Вендорные префиксы — это приставки к названию CSS свойства, которые добавляют производители браузеров для нестандартизированных свойств.
