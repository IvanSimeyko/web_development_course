Подключение js:
1. тег <script>. Страница полностью выполняет скрипт когда загружается, поэтому возможны задержки.
2. Подключение внешнего js файла с помощью тега <script>. Чтобы браузер не тормозил, рекомендуется тег <script> добавлять в конце html-документа (если, конечно, нет функционала, который надо выполнить до загрузки html-страницы. В таком случае нужно подклюить в <head>)

Важно:
- каждая строка кода - инструкция для браузера;
- ; означает конец инструкции
- регистрозависимость.


Типы данных (примитивные):
число
   - Nan (Not a number)
   - infinity (бесконечность)
строка
булевый тип
специальное значение "null" (переменная создана, но ее значением пока неизвестно)
специальное значение "undefined" (значение не присвоено)

Массивы:
arr.unshift() - добавляет элементы в начало массива.
arr.shift() - удаляет элементы в начало массива.
arr.join() - массив переделать в строку

Инкремент (++) увеличивает переменную на один
Декремент (--) уменьшает переменную на один.

Когда пишем === (!==) проверяем не только на равность значений, но и на равность типов.

Взаимодействие с пользователем:
allert, promt(если перед promt поставить +, то введенное пользователем значение будет приведено к типу число), confirm

Циклы. 
Итерация - повторение цикла.
