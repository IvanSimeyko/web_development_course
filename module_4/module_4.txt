Функции
Функция блок кода, который определяется один раз и может использоваться многократно.
Если не знаем прийдет в функции аргумент или нет, то можем записать так:
var num_1 = num_1 || 1;

Объявление функций:
1. function declarations: function name() { body ... } Функция в таком случае может быть вызвана до того как будет объявлена.
2. function expression: var name = function() {body...} Функция в таком виде не может быть вызвана до того как объявлена.
