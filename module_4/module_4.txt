Функции
Функция блок кода, который определяется один раз и может использоваться многократно.
Если не знаем прийдет в функции аргумент или нет, то можем записать так:
var num_1 = num_1 || 1;

Объявление функций:
1. function declarations: function name() { body ... } Функция в таком случае может быть вызвана до того как будет объявлена.
2. function expression: var name = function() {body...} Функция в таком виде не может быть вызвана до того как объявлена.
3. Функция объявляется и сразу вызывается, т.е. имя функции не нужно, т.к. она выполняется сразу после запуска программы:
(function() {
   body ...
}();
Следует помнить, что имя можно все-таки задать, если требуется запустить функцию рекурсивно.

4. var f = new Function('x', 'y', 'return x+y');

Переменные функции:
локальные - переданные в аргументах или объявленные внутри функции при помощи var=... Данные переменные уничтожаются после завершения функции. 

Если переменную в середине функции объявить без ключевого слова var, то она автоматически станет глобальной и перезатрет глобальную переменную с таким же именем (если конечно она есть). 

Глобальную область видимости, определенную для программы в целом называют Window.

Глобальные переменные засоряют память, поэтому по возможности нужно использовать их поменьше.

DOM in js.
Когда загружена html-страница браузер создает DOM - объектную можель документа. Т.о js имеет полный доступ ко всем документам на странице и может:
-изменять элементы на странице;
- изменять атрибуты и CSS-стили;
- удалять элементы и атрибуты;
- добавлять новые элементы и атрибуты;
-реагировать на действия полльзователя относительно HNML-документов;
-создавать новые действия.

Все стили, которые в СSS пишется через дефис в js пишутся через camelCase, например: nameVar.style.fontSize = '30px';

HTML DOM - методы.
Это действия которые мржем применть к html-документам. 
innerHTML - меняет текст в элементе. 
document.querySelector('input[type='text']').values; - получить значение введенное в поле формы.
